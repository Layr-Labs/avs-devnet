# AvsDevnet CLI

A wrapper over the [Kurtosis CLI](https://docs.kurtosis.com/cli).

## Building the program

```sh
make deps     # installs dependencies
make install  # installs the project
```

## Starting the devnet

```sh
devnet run -c <config-path>
```

## Stopping the devnet

```sh
devnet clean -c <config-path>
```
